# Agentic Boilerplate

A comprehensive boilerplate for building projects powered by code agents (Claude, GPT, etc).

## Overview

This repository provides a structured foundation for developing applications that leverage AI code agents. It includes documentation templates, configuration guidelines, and best practices for agent-driven development.

## Quick Start

1. Clone this repository
2. Read through the documentation files to understand the structure
3. Use `.cursor/rules.md` to configure your IDE agent rules
4. Start building in the `src/` directory

## Documentation

- **[CLAUDE.md](./CLAUDE.md)** - Guidelines for working with Claude and AI agents
- **[SPEC.md](./SPEC.md)** - Project specification template
- **[TRACE_MODEL.md](./TRACE_MODEL.md)** - Model tracing and debugging documentation
- **[DECISIONS.md](./DECISIONS.md)** - Architectural decisions and rationale
- **[ROADMAP.md](./ROADMAP.md)** - Project roadmap and milestones
